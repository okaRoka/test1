・cmdインストール
yarn → npm install -g yarn

React → yarn global add create-react-app

・PATHを通す
1/「yarn global bin」コマンドを打ち「C:~」をコピー
2/Winキー、環境変数、Enter
3/Pathを選択、編集
4/新規、ペースト、OK
5/パソコン再起動
